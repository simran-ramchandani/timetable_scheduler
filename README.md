# Timetable Scheduler System

This project is a timetable scheduling system developed in Python. The
system is designed to automate the creation and management of academic
timetables while minimizing scheduling conflicts.

It focuses on structured logic implementation and efficient allocation
of subjects, time slots, and possibly faculty or classroom resources.

------------------------------------------------------------------------

## Project Overview

The Timetable Scheduler aims to:

-   Automatically generate timetables
-   Prevent overlapping time slots
-   Allocate subjects efficiently
-   Improve scheduling accuracy
-   Reduce manual scheduling effort

The project demonstrates algorithmic thinking and problem-solving
applied to academic scheduling.

------------------------------------------------------------------------

## How to Run

1.  Ensure Python is installed.

2.  Navigate to the project directory.

3.  Run the program:

        python last_running_v3.py

    or

        python ui_improved.py
